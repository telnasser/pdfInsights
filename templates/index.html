Key Features

Advanced Document Processing

Intelligent chunking of PDF documents using sentence-based, paragraph-based, or sliding window strategies with configurable chunk sizes and overlap.

Semantic Search & Reranking

Powerful vector-based search with reranking capabilities, delivering the most relevant document chunks for your queries.

Persistent Storage

All documents, chunks, and queries are stored in a PostgreSQL database for reliable access and historical analysis.

Visualization Tools

Interactive visualizations for document chunks, showing relationships and similarity between different parts of your documents.

How It Works

  1. Upload a PDF Document
    Upload any PDF document to be processed and indexed.
  2. Document Processing
    The document is analyzed, chunked using your chosen strategy, embedded into a vector space, and stored in the database.
  3. Ask Questions
    Query your documents with natural language questions to get precise, contextual answers.
  4. Explore and Analyze
    Visualize document chunks and their relationships to gain deeper insights.
